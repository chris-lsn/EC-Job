# Freelance Matching System
EC-Job is freelance matching system platform where provided a place for user to find job or freelancer.

**Used Techniques**

Web Front-End: VueJS

Web Back-End: NodeJS, MongoDB

Mobile (Hybrid App): VueJS + Cordova
